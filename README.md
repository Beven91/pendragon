# pendragon

## 一、简介

`React SPA`开发案例

使用`webpack`搭建的脚手架

###### 支持哪些特性?

- 支持`es6` 与 `es7` 等语法
- 支持图片压缩
- 支持热更新
- 支持响应式布局&样式兼容(`autoprefix`)
- 支持代码按需加载
- 支持路由`代码拆分`
- 支持`微信`接入
- 支持`数据打点`
- 支持`前端异常上报`
- 集成`redux`
- 集成[`ant-design-mobile`](https://github.com/ant-design/ant-design-mobile/)组件库

&#8195;
